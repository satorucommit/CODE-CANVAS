# CodeCanvas

A web-based application for organizing notes and code snippets with multiple templates and client-side storage.

## Features

- Multiple template types (Notes, Checklists, Timelines, Diagrams, Brainstorm, Code Snippets)
- Syntax highlighting for code storage
- Color-coded organization
- Search and filter functionality
- Local browser storage with export/import capabilities
- Responsive design
- No login required

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser

### Usage
- Create items using templates
- Content is automatically saved to browser storage
- Export/import data via Settings menu

## Technical Stack

- Client-side storage using localStorage
- No backend or authentication required
- JSON export/import format
